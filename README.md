# Dahub — Documentación Técnica

Dahub centralizador de datos.

Dahub es un sistema web diseñado para unificar en un solo lugar toda la información operativa que una empresa necesita para tomar decisiones rápidas y con contexto. Integra datos de múltiples fuentes —telemetría, geocercas, eventos, conductores, unidades, video y más— para convertirlos en herramientas reales de monitoreo, análisis y automatización.

La plataforma se organiza en módulos pensados para cubrir todo el ciclo operativo:

User Admin: Gestión de usuarios, roles, permisos y seguridad.

Data: Administración de activos, conductores, eventos externos e internos, zonas, rutas y notificaciones.

Safety: Dashboards, Control Tower para monitoreo en tiempo real, tracking en vivo, histórico de viajes, mapas de calor, galería de imágenes y videos generados por eventos.

Productivity: Análisis de tiempos y estados de las unidades dentro y fuera de geocercas.

Maintenance: Salud de unidades, fallas, alertas y diagnósticos operativos.

El objetivo es simple: dar una visión completa y accionable de toda la operación en un solo panel, eliminando silos de información y permitiendo decisiones más rápidas, precisas y con menos fricción.
